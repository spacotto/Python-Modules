# Python-Modules

This curriculum comprises 11 Python modules that progressively build programming expertise. 

- Modules 00-04 establish fundamentals: syntax, functions, classes, data structures, and file I/O.
- Module 05 introduces polymorphism and method overriding for flexible object-oriented design.
- Module 06 covers the import system, package organisation, and circular dependency management.
- Module 07 explores abstract base classes, multiple inheritance, and design patterns (Factory, Strategy) for enterprise architecture.
- Module 08 teaches professional development workflows: virtual environments, pip/Poetry package management, and secure configuration with environment variables.
- Module 09 focuses on runtime data validation using Pydantic models with custom validators.
- Module 10 completes the journey with functional programming: lambda expressions, higher-order functions, closures, decorators, and the functools module.

Together, these modules provide a complete coverage of procedural, object-oriented, and functional programming paradigms essential for modern Python development.
